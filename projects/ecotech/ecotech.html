Galeria do Projeto

Principais Funcionalidades

Pagamentos via PIX

Integração robusta com a API do Mercado Pago para processar assinaturas de planos de forma segura e automatizada via PIX.

Planos para Empresas

Oferece serviços exclusivos para empresas, como coleta programada no local e emissão de certificados de descarte ecológico.

Soluções para Usuários

Permite que usuários comuns encontrem pontos de coleta próximos ou agendem a retirada de seus equipamentos eletrônicos.

Confirmação de Assinatura

Páginas detalhadas para que o usuário confirme o plano escolhido antes de prosseguir para o pagamento.

Geração de QR Code

O sistema gera dinamicamente o QR Code e o código "copia e cola" do PIX após a confirmação do plano pelo usuário.

Gestão de Assinantes

O backend atualiza o status do plano do usuário ou empresa no banco de dados assim que a assinatura é iniciada.
