Principais Funcionalidades

Login Social com Google

Sistema OAuth 2.0 seguro para cadastro e login rápido, melhorando a experiência do usuário e a segurança.

Recuperação de Senha

Fluxo completo de redefinição de senha com validação via token enviado por um servidor de e-mail SMTP.

Hub de Conteúdo Protegido

Área de membros com autenticação de sessão, garantindo que apenas usuários pagantes acessem conteúdos exclusivos.

Pagamentos com Mercado Pago

Integração com a API do Mercado Pago para gerenciar assinaturas e pagamentos de forma segura e transparente.

Painel Administrativo

Template AdminLTE integrado ao backend para uma gestão completa de usuários, conteúdos, assinaturas e relatórios.

Design Responsivo

Interface construída com Bootstrap, totalmente adaptável para uma experiência de usuário consistente em desktops e celulares.
